# 테트리스 (Tetris)

브라우저에서 바로 플레이할 수 있는 클래식 테트리스 게임입니다.  
별도의 설치나 빌드 과정 없이 HTML, CSS, JavaScript만으로 동작합니다.

## 실행 방법

### 방법 1: 파일 직접 열기
`index.html` 파일을 브라우저에서 더블클릭하여 엽니다.

### 방법 2: 로컬 서버 (권장)
```bash
# Python 3
python -m http.server 8000

# Node.js (npx)
npx serve .
```
브라우저에서 `http://localhost:8000` 으로 접속합니다.

### 방법 3: GitHub Pages
저장소를 GitHub에 푸시한 뒤 Settings → Pages에서 배포하면 온라인에서 플레이할 수 있습니다.

## 조작법

| 키 | 동작 |
|---|---|
| `←` `→` | 좌우 이동 |
| `↓` | 소프트 드롭 (빠르게 내리기) |
| `↑` | 블록 회전 |
| `Space` | 하드 드롭 (즉시 착지) |
| `C` | 홀드 (블록 보관/교환) |
| `P` | 일시정지 |
| `Enter` / `Space` | 게임 시작 / 재시작 |

## 게임 규칙

- 7가지 테트로미노(I, O, T, S, Z, J, L)가 무작위로 등장합니다.
- 가로줄이 가득 차면 해당 줄이 사라지고 점수를 얻습니다.
- 10줄을 클리어할 때마다 레벨이 올라가며 블록 낙하 속도가 빨라집니다.
- 블록이 더 이상 쌓일 수 없으면 게임 오버입니다.

## 점수 시스템

| 클리어 줄 수 | 기본 점수 (× 레벨) |
|---|---|
| 1줄 | 100 |
| 2줄 | 300 |
| 3줄 | 500 |
| 4줄 (테트리스) | 800 |

- 소프트 드롭: 줄당 1점
- 하드 드롭: 줄당 2점

## 프로젝트 구조

```
cursor-tetris/
├── index.html          # 메인 HTML
├── style.css           # 스타일
├── js/
│   ├── tetrominoes.js  # 블록 정의 및 회전
│   ├── board.js        # 보드 로직 (충돌, 라인 클리어)
│   └── game.js         # 게임 엔진, 렌더링, 입력
└── README.md
```
